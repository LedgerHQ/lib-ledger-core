Detailed Description

Structure representing DER encoded signature DER format :

  • DER prefix
  • Length of rest of signature
  • Marker for r value
  • Length of r value
  • r value, Big Endian
  • Marker for s value
  • Length of s value
  • s value, Big Endian
  • SIGHASH byte (ALL, NONE, SINGLE)
